# Install

### Run server

```
npm install
```

```
npm start
```

Server will run on [http://localhost:4444](http://localhost:4444)

### Run client

```
cd client
```

```
npm install
```

```
npm start
```

The app will run on [http://localhost:3000](http://localhost:3000)
